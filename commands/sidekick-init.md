---

Run the Sidekick setup now using the **sidekick-init** skill.

Walk the user through the multiple-choice questionnaire (role, chat language,
default output language, and email/messages/storage/calendar connections), then
— after a summary and confirmation — write `sidekick.settings.md` to the Cowork
root, scaffold at least one project, and create the `_triage/` and
`_archive/projects/` system folders. Close by explaining how to schedule the
triage as a Cowork scheduled task and which connectors to enable. Init records
intent only; it never turns a connector on.
